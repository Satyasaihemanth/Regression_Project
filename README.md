# 💻 Laptop Price Prediction

## 📌 Overview
This project predicts the price of a laptop based on its specifications using Machine Learning.

## 🚀 Features
- Predict laptop price based on input features
- Uses regression model
- User-friendly interface (Streamlit)

## 🛠 Tech Stack
- Python
- Scikit-learn
- Pandas, NumPy
- Streamlit

## 📂 Project Structure
- backend.py → Model prediction logic
- frontend.py → Streamlit UI
- model.pkl → Trained model
- dataset.csv → Training data

## ⚙️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run app:
streamlit run frontend.py

## 📊 Input Features
- Company
- RAM
- CPU
- GPU
- Storage
- Screen Size
- Weight

## 🎯 Output
Predicted Laptop Price 💰
